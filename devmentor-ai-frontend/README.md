# 🚀 DevMentor AI

> AI-Powered Developer Assistant for Students & Developers

DevMentor AI is a modern web application that helps developers analyze, understand, optimize, and improve their code. It provides an interactive code editor, AI-powered analysis interface, GitHub repository analyzer, interview preparation, code conversion, reports, and much more.

> ⚠️ Current Version: Frontend Only (Backend APIs Coming Soon)

# 🚀 Installation

Clone the repository


Move into project

```bash
cd devmentor-ai
```

Install dependencies

```bash
npm install
```

Start development server

```bash
npm run dev
```

Build production

```bash
npm run build
```

Preview production build

```bash
npm run preview
```
